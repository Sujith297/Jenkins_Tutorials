prerequisites
1) Install git 
command:-  yum install git 

Steps to create manual job to build 
1) select create item option 
2) give name to job as you like ex. project_name or service_name
3) select type of job (pipeline , freestyle, multipipeline) here we select freestyle
4) next configuire source code management select git
5) get your repo https link and paste
6) specify git branch
7) Save the job and build it 

observe:-
1) console output (location of repo)
2) workspace (check files)
